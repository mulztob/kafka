# Kafka Demo with Stencil and ...

## Getting Started

## Configuration

## Kafka Cluster `kafka-cluster`

- `docker-compose up -d`

## consumer/producers `kafka-node`

- npm i
- configure `.env`
- npm run producer
- npm run consumer

## UI `stencil-ui`

Example App in Stencil.js which consumes some kafka topic(s) and show something in UI

- npm i
- configure `.env`
- npm start
